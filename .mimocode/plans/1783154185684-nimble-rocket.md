# Plan: Replace canvas stars with CSS scroll-based parallax starfield

## Context

The CodePen uses CSS `box-shadow` to render hundreds of star points across 3 depth layers. Your current project uses a `<canvas>` with JS-based twinkling. The user wants **scroll-based parallax**: star layers shift at different speeds as the user scrolls the page, not continuous animation.

**Goal:** Replace the canvas stars with CSS-based star layers (using the CodePen's box-shadow technique) that move at different parallax rates on scroll.

## Changes

### 1. `src/index.html` — Replace `<canvas>` with parallax star divs

Replace `<canvas id="stars"></canvas>` with:
```html
<div class="stars-wrapper">
  <div id="stars-layer1"></div>
  <div id="stars-layer2"></div>
  <div id="stars-layer3"></div>
</div>
```

### 2. `src/styles.css` — Add parallax star CSS

- Remove the existing `#stars{position:fixed;inset:0;z-index:0;pointer-events:none}` rule (line 36)
- Add `.stars-wrapper`: `position: fixed; inset: 0; z-index: 0; pointer-events: none; overflow: hidden`
- Add `#stars-layer1`, `#stars-layer2`, `#stars-layer3` with:
  - `position: absolute; top: 0; left: 0; width: 1px; height: 1px; background: transparent`
  - `box-shadow` with the star coordinates from the CodePen (3 layers: 1px, 2px, 3px stars)
  - `will-change: transform` for GPU-accelerated parallax
  - `pointer-events: none`
- No CSS animation — position will be controlled by JS scroll handler

### 3. `src/modules/stars.ts` — Rewrite as scroll parallax controller

Replace the canvas logic with a lightweight scroll handler:
- On scroll, apply `transform: translateY(scrollY * rate)` to each layer
- Layer 1 (small stars): rate = -0.05 (moves slowly, far away)
- Layer 2 (medium stars): rate = -0.1 (medium speed)
- Layer 3 (large stars): rate = -0.2 (moves fastest, close)
- Use `requestAnimationFrame` for smooth performance
- Include `prefers-reduced-motion` check: disable parallax if user prefers reduced motion
- Handle `visibilitychange` to pause/resume

### 4. `src/main.ts` — No changes needed

The `initStars()` import and call remain as-is since `stars.ts` is rewritten in-place.

## Files to modify
