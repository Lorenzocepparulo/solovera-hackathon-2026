# UI/UX Enhancement Plan — Solovera Hackathon Landing Page

## Goal
Make the website clearer and more polished: better visual hierarchy, more breathing room, and refined component styling — while keeping the space/rocket theme fully intact.

## Files to Modify
- `src/styles.css` — all visual changes (primary file)
- `src/index.html` — one minor accessibility improvement (optional)

No changes to any TypeScript modules.

---

## Design Principles
1. **3-tier visual hierarchy**: headings > body > tertiary (eyebrows, labels) — currently everything competes at the same weight
2. **Consistent spacing tokens**: introduce CSS variables for section gaps, content gaps, card radius, and transitions
3. **Subtle depth**: add hover box-shadows to cards for a sense of elevation
4. **Better readability**: increase line-heights on body text, improve label/input spacing in forms

---

## 1. CSS Variables (`:root`)

Add to the existing variable block:
```css
--section-gap: 140px;
--section-gap-mobile: 80px;
--content-gap: 48px;
--card-radius: 24px;
--transition-base: .25s cubic-bezier(.4,0,.2,1);
```

Also widen the container: `max-width: 1120px`, `padding: 0 32px`.
